# Viral Content Rules

The taxonomy and platform physics behind /og-viral.

## Hook taxonomy (10 categories — every campaign samples all 10)

| Category | Pattern skeleton |
|---|---|
| Contrarian | "Everyone tells you X. X is why you're stuck." |
| Pain-driven | Name the exact expensive/embarrassing moment they had this week |
| Proof-driven | Specific result + specific timeframe + specific mechanism |
| Mistake-based | "The [role]'s mistake that costs $X — and the 10-minute fix" |
| Status/authority | Insider framing: "After [credential/scale], here's what nobody says out loud" |
| Speed/result | Compressed timeline: "X in Y days without Z" (only when true) |
| Risk-reversal | Lead with the guarantee/de-risk before the pitch |
| Curiosity | Open loop with a concrete noun, not vague bait |
| Industry secret | "The thing vendors charge $X for is actually…" |
| Cost-of-inaction | Price the status quo: what waiting costs per month, in their units |

First 2 seconds / first line carries the hook verbatim. If the hook needs the third sentence to land, it isn't a hook.

## Platform specs

| Platform | Format center of gravity | Notes |
|---|---|---|
| LinkedIn | 1,300–2,000 char post, line-broken, no external link in body (comment it) | Founder-led performs; document > announce |
| X | Thread: hook tweet + 5–9 beats + CTA tweet | Hook tweet must stand alone |
| Instagram | Reels script 15–45s + caption w/ first-line hook | Caption first line = hook, hashtags minimal |
| TikTok/Reels/Shorts | 15–45s vertical, hook ≤2s, captions on | Pattern-interrupt visual in first frame |
| YouTube long | Title (Edge Mandate: attention-earning, honest) + hook-front description + chaptered outline | Titles A/B in threes |
| Email | Subject = hook category match; body = one idea, one CTA | Plain-text-feel outperforms design-heavy for DR |
| Blog/SEO | Answer-first structure, schema-ready | Repurpose from winning long-form |
| PDF lead magnet | Solves ONE painful step completely; CTA to the rung that solves the rest | Title uses the winning hook |

## Repurposing matrix (minimum path per winning asset)

Long-form (YT/blog) → 3 shorts → 2 LinkedIn posts → 1 X thread → 1 email → 1 lead-magnet section. Winners get repurposed before new ideas get funded.

## Discipline

- Every asset carries the mapping footer (buyer/pain/offer/CTA/destination/GHL tag/pipeline outcome) — modes/06-viral.md.
- Read `winners-library.md` before generating; imitate your own winners first, competitors' second, novelty last.
- No engagement bait, no fabricated stories presented as real, no trend-jacking that violates the profile's forbidden-topics list.
- Approval queue before anything is scheduled or posted. Always.
