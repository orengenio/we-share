# Edge Engine

The enforcement layer of the **Edge Mandate**: nothing generic ships. Every asset — copy, script, hook, page, brief, offer name — clears this file BEFORE Brand Guard. Brand Guard makes it on-brand; the Edge Engine makes it **unmistakable**. Generic is a defect class, not a style choice.

## 1. Voiceprint Forge (build once per engagement, evolve forever)

Inputs: the founder's actual speech (transcripts, posts, voice notes) + the brand lexicon + the personas' voice-of-customer banks. Output — stored in `profile.brand.voiceprint`:

```
signature moves: <3–5 recurring constructions this brand owns —
  e.g., OrenGen's long-setup/short-punch rhythm; the named-enemy turn; the receipts-first claim>
named mechanisms: <every proprietary term in play — methods, formats, offers>
enemy + stakes: <what this brand is AGAINST, in one line — OrenGen's: dependency>
rhythm profile: <sentence-length pattern, punch placement, humor register>
never-say swaps: <generic phrase → the house way of saying it>
```

Winning lines from /og-optimize get promoted into signature moves. The voiceprint compounds.

## 2. The Slop List (banned on sight, all customer-facing output)

**Words as filler:** game-changer · unlock · elevate · revolutionize · seamless · effortless · supercharge · next-level · harness · delve · robust · streamline (as a verb of vibes) · "solutions" as a noun of nothing — plus everything on Brand Guard's forbidden list.

**Constructions:** "It's not just X, it's Y" · rhetorical-question openers ("Tired of…?") · "Imagine a world where…" · unearned triad stacks ("Faster. Smarter. Better.") · "Here's the thing:" as a crutch · "Unpopular opinion:" openers · broetry line breaks doing the work the idea should do · "In today's [adjective] landscape…"

**Structures:** the interchangeable 5-tips listicle · the summary-sandwich that says everything twice · the case study with no numbers · the About page written about the company instead of the customer.

The list is additive per engagement — every slop pattern found in a competitor's copy gets logged and banned locally. Their clichés are our contrast.

## 3. The Four Tests (every asset, pass/fail each)

1. **Swap Test** — replace the brand name with the top competitor's. If the asset still reads true, it fails. Rewrite until only this brand could have said it.
2. **Only Test** — can the core claim honestly begin with "Only [brand]…"? If a competitor could claim it too, sharpen the mechanism, the proof, or the framing until they can't.
3. **Bar Test** — would the founder say this sentence out loud to a smart friend? If it only survives in a slide deck, it dies here.
4. **Screenshot Test** — is there at least one line worth screenshotting, quoting, or stealing? If nothing is quotable, nothing is memorable.

## 4. Edge Score (1–10, ships at ≥7)

| Component | Points |
|---|---|
| Voiceprint adherence (signature moves present, rhythm matches) | 0–3 |
| Four Tests passed (1 pt each) | 0–4 |
| Named mechanism present where a method is described | 0–1 |
| Enemy/stakes present (the asset knows what it's against) | 0–1 |
| Specificity density (numbers, proper nouns, timeframes ≥3 per ~100 words of claim copy) | 0–1 |

Score <7 → returned for rewrite with the failing components named. Score logged in STATE next to the Brand Guard pass. No warnings-and-ship — the Edge Gate is a gate.

## 5. Standing doctrines

- **Name everything.** Methods, formats, frameworks, offers — proprietary names (™-candidates). Unnamed = rented language; named = owned IP.
- **Every asset knows its enemy.** Copy without stakes defaults to benefit-mush. OrenGen's enemy is dependency; every Mode C engagement defines its own in the voiceprint.
- **The market writes the first draft.** Voice-of-customer lines beat copywriter cleverness; arrange their words, then add the edge.
- **Contrast beats volume.** One asset that could only be this brand outworks ten that could be anyone.
