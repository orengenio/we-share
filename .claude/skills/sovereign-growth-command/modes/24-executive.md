# Mode 24 — /og-executive

The Fractional Executive Machine — the C-suite advisory lane (fractional CEO / CTO / COO / CMO) run as a system. Positioning doctrine: **not a tool guy — the executive operator who brings strategy, technology, and implementation together.** The premium framing: C-suite AI transformation advisor. Masters modeled: Charan (execution), Martin (strategy), Goldsmith (behavior) — see `references/masters-library.md`.

## The consulting posture (Charan law)

**Do not sell ideas. Diagnose bottlenecks, simplify priorities, assign accountability, and drive execution.** The engagement never opens with a pitch — it opens with a diagnosis: Problem clarity → Root cause → Talent alignment → Operating cadence → Accountability → Execution → Measurable outcome.

## The engagement OS

```
Executive Diagnosis → Strategic Choice → Operating Model → AI / Tech Enablement
→ Leadership Behavior Gaps → Execution Cadence → Scorecard → Accountability Review
```

## Strategic choice (Martin law)

Strategy is a choice cascade, not a planning document:

```
Winning aspiration → Where to play → How to win → Capabilities required → Management systems required
```

**A strategy that does not force tradeoffs is not strategy — it is wishful planning.** Every engagement (and OrenGen itself) names the specific markets, specific offers, specific channels, and what it is explicitly NOT doing. The not-doing list is the strategy's proof of life. This cascade also governs /og-structure's model work — same doctrine, engagement-level application here.

## Behavior layer (Goldsmith law)

Executives don't only need strategy — they need **behavior change, communication discipline, decision hygiene, and accountability systems.** The loop: Feedback → Behavior gap → Stakeholder input → Action commitment → Follow-up → Measured improvement. Feedforward over autopsy: ask stakeholders what to do *next*, not what went wrong. What made the executive successful can cap their next level — name the capping behavior early and kindly.

## The four seats (diagnostic lens per fractional role)

| Seat | First diagnostic pass |
|---|---|
| **CEO** | Strategy cascade completeness · execution cadence · decision bottlenecks · accountability map |
| **CTO** | Stack + **dependency audit** (rented vs. owned — the brand-enemy lens as engineering review) · security/compliance posture · build-vs-buy discipline |
| **COO** | Process coverage + founder-dependence → hands directly to /og-systems |
| **CMO** | Growth engine audit → SOVEREIGN itself IS the CMO seat's toolkit: genesis→persona→offer→funnel→viral→optimize |

## Operating cadence (the deliverable that makes retainers stick)

Weekly execution huddle agenda · monthly scorecard (3–7 metrics max, owner per metric, red-line per metric) · quarterly strategy review against the choice cascade · accountability map (every priority has ONE name on it). Cadence documents ship as templates the client keeps — the value compounds even when the engagement ends, which is exactly why they renew.

## Money + gates

Retainer architecture per /og-offer (setup + monthly, scope guard: fractional ≠ unlimited — the scope doc lists included cadences and named deliverables). Strategic recommendations with >$10k client exposure route through /og-board framing before delivery. Credentials truthful always: advisor and operator — never licensed titles not held (Brand Guard never-claim list governs).
