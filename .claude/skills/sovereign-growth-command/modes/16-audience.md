# Mode 16 — /og-audience

The audience-growth engine. /og-viral makes conversion content (every asset maps to a funnel); this mode grows the **asset that makes conversion cheaper: the audience.** Both run under the same approval gates and Edge Engine — growth content that sounds like everyone else grows nothing.

## The split (keep it clean)

| | Audience content | Conversion content (/og-viral) |
|---|---|---|
| Optimizes for | reach, saves, shares, follows | clicks, opt-ins, purchases |
| CTA | follow / subscribe / save / reply | funnel destination |
| Ratio | ~70–80% of output | ~20–30% of output |

## Growth mechanics (generate per active platform)

1. **Profile optimization** — bio formula: who it's for + the result + the proof + one CTA. Pinned asset = best converting piece. Name/handle searchable for the niche term.
2. **Cadence engine** — platform-band frequency plan the operator can actually sustain. Consistency beats volume; a cadence you quit is a cadence that never existed.
3. **Format ladders (follow-drivers per platform)** — X: threads + high-value replies under big accounts; LinkedIn: document posts + a daily comment strategy done with dignity; TikTok/Reels/Shorts: series formats + first-2-second retention; YouTube: packaging (title + thumbnail) outranks production; email: the owned-channel flywheel every platform feeds.
4. **Named franchise formats** — recurring segments the audience returns for ("<Named Segment>, every Tuesday"). The named-mechanism rule applies to formats: own a show, not just posts.
5. **Engagement windows** — the 30 minutes before/after posting spent in replies; reply strategy = add the missing insight, never "Great post!"
6. **Collab plays** — duets/stitches, guest threads, newsletter swaps, podcast circuits — matched to the persona's watering holes.
7. **The off-ramp (non-negotiable)** — every platform is rented land. Every growth play includes a migration path to owned channels: follower → email/SMS list → community. This is the brand-enemy doctrine applied to audience: never let the algorithm hold the relationship hostage.

## Metrics that matter (leading, not vanity)

Non-follower reach % · saves + shares per post · profile-visit → follow rate · follower velocity (7/30-day) · list captures per 1k followers. Follower count alone is a lagging trophy; these predict it.

## Rules

- Reads persona watering holes + winners-library before planning; imitates own winners first.
- All scheduled posting goes through the approval queue. No auto-posting, ever.
- Platform tactics respect the compliance classification table — no engagement pods, no follow/unfollow botting, no bought followers. Fake audience is negative equity.

## Produce (`artifacts/<slug>/audience/`)

`growth-plan-<platform>.md` per platform, `franchise-formats.md`, `cadence-calendar.md`, `off-ramp-map.md`, and monthly velocity reviews feeding /og-optimize.
